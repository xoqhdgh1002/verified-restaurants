# 🗺️ 지도 기능 빠른 시작 (10초!)

## ✨ **API 키 불필요 - 무료 오픈소스!**

**Leaflet + OpenStreetMap**을 사용하므로 API 키 발급이나 설정이 전혀 필요 없습니다!

## 🚀 사용 방법

### 1. 개발 서버 실행

```bash
npm run dev
```

### 2. 식당 목록 페이지 접속

```
http://localhost:3000/restaurants
```

### 3. 지도 확인! 🎉

1. **"🗺️ 지도"** 버튼 클릭
2. 식당 마커 확인!
3. 마커 클릭으로 상세 정보 보기

**끝!** 😊

---

## ✨ 주요 기능

### 📍 내 위치 기반 검색
1. "📍 내 위치" 버튼 클릭
2. 브라우저 권한 허용
3. 정렬을 "거리순"으로 변경
4. 가까운 식당부터 표시됨!

### 🔍 지도에서 식당 찾기
- **녹색 마커(✓)**: 검증 완료 식당
- **빨간 마커(📍)**: 검증 대기 식당
- 마커 클릭으로 정보 확인

## 🐛 문제 해결

### Q: 지도가 안 보여요
- 인터넷 연결 확인
- 서버 재시작 (`Ctrl+C` 후 `npm run dev`)
- 브라우저 콘솔 확인 (F12)

### Q: 위치 정보를 가져올 수 없어요
- 브라우저 설정 → 위치 권한 허용
- HTTPS 환경에서만 작동 (localhost는 예외)
